Para baixar um site com wget execute o seguinte comando: 
wget -r --no-parent --html-extension --convert-links http://site.com
Descrição dos parametros:
-r = baixar recursivamente
--no-parent = não fazer download de links externos que estejam no site
--html-extension = salvar as páginas presentes no site em html
--convert-links = converter os links para os html acima sem endereço
